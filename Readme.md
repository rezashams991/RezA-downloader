📢 آموزش بسیار ساده و رایگان: دانلود هر فایل 

 با این ترفند ساده می‌توانید فایل‌های خود را دانلود کنید.  این روش تا بیستو یک گیگ تست شده و بدون مشکل جواب داد




3️⃣ اجرای عملیات دانلود:
   - روی تب «Actions» کلیک کنید.
   - در سمت چپ، روی «دانلود فایل دلخواه شما» کلیک کنید.
   - دکمه آبی «Run workflow» را بزنید.
   - یک کادر باز می‌شود. در آن، لینک مستقیم فایل مورد نظر خود و رمز مورد نظر را جایگذاری کنید.
   - روی دکمه سبز «Run workflow» کلیک کنید.

4️⃣ دریافت فایل دانلود شده:
   - بعد از چند دقیقه تا چند ساعت بسته به حجم فایل، کنار کلمه run یک دایره سبز ظاهر میشود به این معنا که فایل مورد نظر دانلود به صورت تکه تکه در مخزن اصلی ذخیره شده (لزومی نداره در صفحه بمونید میتونین بعد از آغاز فرایند ببندین صفحه رو و دوباره به صفحه برگردید)
   - در صورت نیاز برای بررسی میزان پیشرفت میتونین روی download-and-split کلیک کنین به صفحه توضیحات پیشرفت هدایت میشین و زمان تقریبی دانلود رو اونجا میبینین بعد از اتمام دانلود منتظر اتمام بقیه مراحل بمونین و بعد از اتمامشون ادامه راهنما رو انجا بدین
   - بالای صفحه روی نام مخزن اصلی برای مثال اینجا ( github-file-downloader) کلیک کنید
   - روی دکمه Code کلیک و گزینه دانلود زیپ رو بزنید
   - فایل دانلود شده رو از حالت زیپ خارج کنین
   -فایل فشرده داخل آن با پسوند001 رو با برنامه های استخراج کننده مثل 7z یا winrar باز و استخراج کنین به صورت خودکار تمام تکه های فایل به هم چسبیده و فایل کامل و نهایی آماده است

💡 نکات مهم:
- حتماً از «لینک مستقیم» فایل استفاده کنید .
- تا بیستو یک گیگ تست شد بدون محدودیت حجم مدت زمان فایل بیستو یک گیگ یک ساعتو چهل دقیقه ممکنه به نسبت سرعت سرور فایل، متفاوت باشه
- اگر اینترنت شما پایدار نیست به دلیل نداشتن قابلیت ادامه دهندگی لینک گیت هاب و احتمال قطع فرایند میتونین بجای دانلود یکباره هر تکه از فایل رو به صورت جدا دانلود کنین
